Portafolio

Mi proyecto consistió en la creación de una página web para FerreLeyva, una ferretería. Diseñé un sitio funcional y moderno que incluye un catálogo de productos, un carrito de compras y opciones para filtrar y buscar productos fácilmente. Además, integré un sistema para que los clientes puedan c ontactarse a través de WhatsApp al momento de realizar un pedido. La página fue diseñada pensando en la experiencia del usuario, asegurando que sea intuitiva, accesible y visualmente atractiva
